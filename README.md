# ZenithSui Static Website

This folder is ready to upload to a GitHub repository and publish with GitHub Pages.

## Files

- `index.html` - homepage
- `products.html` - product listing and filters
- `product-detail.html` - gallery, variants, quantity, and add to cart
- `style.css` - full responsive styling
- `script.js` - product data, search, cart, filters, gallery, and UI interactions
- `assets/` - logo and favicon assets

## GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to repository `Settings > Pages`.
4. Set source to `Deploy from a branch`, branch `main`, folder `/root`.
5. Open the published GitHub Pages URL after deployment finishes.

No backend or build step is required.
